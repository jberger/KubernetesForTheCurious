# Kubernetes for the Curious

### Joel Berger

---

## What is Kubernetes?

Short answer:

`docker-compose` but clustered

---


## What is Kubernetes?

Longer answer:

* Container engine
* Networking engine
* Coordination controller
* Distributed-concensus database
* API

---

## Why?

Coordination of containers is:

* Decentralized
* Active/ongoing
* Declarative
* Observable
